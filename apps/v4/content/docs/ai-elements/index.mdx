AI Elements are specialized components designed for building AI-powered user interfaces. These components handle common patterns in AI applications like chat messages, code blocks with syntax highlighting, reasoning displays, task queues, and interactive prompts.

## Available Components

### Conversation Elements
- **Message** - Display chat messages with role-based styling
- **Conversation** - Container for chat message threads
- **Response** - Styled AI response container
- **Suggestion** - Interactive suggestion chips

### Code & Content
- **Code Block** - Syntax-highlighted code display with copy functionality
- **Artifact** - Display generated artifacts
- **Image** - AI-generated image display

### Reasoning & Planning
- **Reasoning** - Display AI reasoning steps
- **Chain of Thought** - Show step-by-step thinking process
- **Plan** - Display structured plans
- **Task** - Individual task display
- **Queue** - Task queue management

### Interactive Elements
- **Prompt Input** - Rich text input with attachments
- **Tool** - Display tool usage and results
- **Actions** - Action buttons and controls
- **Controls** - Media and interaction controls

### Visual Elements
- **Branch** - Show decision branches
- **Node** - Graph node display
- **Edge** - Connect nodes in graphs
- **Canvas** - Drawing canvas for visualizations
- **Connection** - Visual connections between elements

### UI Helpers
- **Context** - Context information display
- **Inline Citation** - In-text citations
- **Loader** - Loading states
- **Shimmer** - Skeleton loading effect
- **Sources** - Reference sources display
- **Open in Chat** - Quick action to open in chat
- **Web Preview** - Preview web content
- **Toolbar** - Action toolbar
- **Panel** - Container panel

## Installation

AI Elements use the same installation process as regular components:

```bash
npx shadcn@latest add [component-name]
```

For example, to add the message component:

```bash
npx shadcn@latest add message
```

## Usage

AI Elements are designed to work together to create complete AI interfaces. They follow the same patterns as other components in the library and are fully customizable.

```tsx
import { Message, MessageContent, MessageAvatar } from "@/components/ui/message"
import { PromptInput, PromptInputTextarea, PromptInputSubmit } from "@/components/ui/prompt-input"

export default function ChatInterface() {
  return (
    <div>
      <Message from="user">
        <MessageAvatar src="/user.jpg" name="User" />
        <MessageContent>Hello, how can you help me today?</MessageContent>
      </Message>
      <Message from="assistant">
        <MessageAvatar src="/ai.jpg" name="AI" />
        <MessageContent>I'm here to help! What would you like to know?</MessageContent>
      </Message>
      <PromptInput onSubmit={(msg) => console.log(msg)}>
        <PromptInputTextarea placeholder="Type your message..." />
        <PromptInputSubmit />
      </PromptInput>
    </div>
  )
}
```

- **Customizable** - Fully themeable with CSS variables
